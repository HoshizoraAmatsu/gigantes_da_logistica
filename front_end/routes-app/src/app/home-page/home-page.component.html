Bem vindo!

Este site tem como objetivo registrar, atualizar e reportar rotas de transporte em um ambiente de guerra, além de calcular e reportar as rotas mais rápidas de um ponto para outro.




Registrar nova rota

Essa funcionalidade serve para adicionar novas rotas ao banco de dados. Todas as rotas devem possuir ponto de origem, ponto de destino, tempo de deslocamento e status da rota

Registrar!
Atualizar rota

Essa funcionalidade tem como objetivo editar rotas já presentes no banco de dados no caso de uma mudança no tempo de deslocamento ou indisponibilidade da rota

Atualizar!
Deletar rota

Essa funcionalidade serve para remover rotas do banco de dados

Deletar!
Encontrar rota

Essa funcionalidade serve para encontrar a rota mais rápida entre dois postos
